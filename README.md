**Smart Home Automation Using ESP32 and Sinric Pro:
**Project Overview:
This project is an IoT-based Smart Home Automation System built using Espressif Systems and Sinric Pro. It allows users to remotely control:
-->Light
-->Fan
-->Automatic Door (Servo Motor)
through a mobile application using Wi-Fi and cloud connectivity.
The system can be operated from anywhere in the world using the Sinric Pro mobile app or voice assistants such as Amazon Alexa and Google Assistant.

Hardware Components:
-->Espressif Systems ESP32 Development Board	1
-->Servo Motor (SG90 or MG996R)	1
-->Light Bulb / LED	1
-->DC Fan	1
-->Jumper Wires	Several
-->Breadboard	1
-->5V Power Supply	1

Software Requirements:
-->Arduino IDE
-->Sinric Pro Account
-->ESP32 Board Package by Espressif
-->Libraries:
-->WiFi.h (ESP32 Core)
-->SinricPro Library
-->ESP32Servo Library
